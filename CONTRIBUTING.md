## クイックスタート

### 1. リポジトリをフォーク・クローン

```bash
git clone https://github.com/your-username/VehicleVision.Pleasanter.ItemsAttachmentsDownloader.git
cd VehicleVision.Pleasanter.ItemsAttachmentsDownloader
```

### 2. ブランチを作成

```bash
git checkout -b feature/your-feature-name
```

### 3. 変更を実装

変更を実装してください。

### 4. コミット・プッシュ

```bash
git add .
git commit -m "feat: 変更の概要"
git push origin feature/your-feature-name
```

### 5. プルリクエストを作成

GitHub上でプルリクエストを作成してください。

## コミットメッセージ

以下のプレフィックスを使用：

| プレフィックス | 用途               |
| -------------- | ------------------ |
| `feat:`        | 新機能             |
| `fix:`         | バグ修正           |
| `docs:`        | ドキュメント更新   |
| `refactor:`    | リファクタリング   |
| `test:`        | テスト追加・修正   |
| `chore:`       | その他（ビルド等） |
